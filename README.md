# 📸 PhotoPost

O PhotoPost é uma aplicação de compartilhamento de fotos e vídeos desenvolvida com foco em aprendizado, arquitetura limpa e boas práticas de desenvolvimento.  O projeto simula uma rede social simplificada, onde usuários podem se cadastrar, autenticar-se e publicar conteúdos multimídia.

Inicialmente:
- Usuários podem se cadastrar e autenticar
- Postar fotos e vídeos
- Visualizar apenas suas próprias postagens
  
Futuramente:
- Integração entre usuários
- Pesquisa e adição de amigos
- Visualização de publicações de outros usuários
- Interface moderna em React
